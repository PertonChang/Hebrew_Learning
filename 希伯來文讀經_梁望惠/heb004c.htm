第四章  詩篇136篇

生詞

          

v 表示第幾節，# 表示錄音帶未錄此字，* 表示動詞的這個形式在聖經中不存在；最後面的 0,50, 100, 200, 500分別代表它在聖經中出現 0-50, 50-100, 100-200, 200-500, 500次以上；以下的詞類變化是為了方便練習解答查閱用，讀生詞時可先略過不看

     
v1 動詞，「唱歌」。 是 Qal 完成式 3 單陽；* 是 Qal 完成式 3 複；* 是 Qal完成式 1單； 是 Qal 未完成式 3 單陽； 是 Qal 祈使式複陽； 是 Qal 不定詞附屬形。50
# 陽性名詞，「主人」。 和 都是複陽加上 1 單詞尾，後者多用在指「主上帝」。500
v3 # 動詞，「照顧、詢問」。 是 Qal 主動分詞單陽獨立形；* 是 Qal 主動分詞單陰附屬形；*是 Qal 主動分詞複陽獨立形； 是 Qal 主動分詞複陽附屬形；* 是 Qal 主動分詞單陽加 3 單陽詞尾。100
動詞，「做」。 是 Qal 主動分詞單陽獨立形； 是介系詞 加 Qal 主動分詞單陽附屬形。500
* 動詞，Nif'al「是奇妙的」。這個動詞沒有 Qal 的形式；* 是 Nif'al 分詞單陰； 是 Nif'al 分詞複陰獨立形； 是 Nif'al 分詞複陰附屬形。50
副詞，「唯獨」。 係加上 3 單陽詞尾。100
v4 陰性名詞，「智慧」。 是介系詞 加名詞單陰。0
介系詞，「在、用、藉著」。500
v5 * 動詞，「建立、鋪張」。 是 Qal 主動分詞單陽獨立形； 是介系詞 加 Qal 主動分詞單陽附屬形。0
# 專有名詞，人名，「約書亞」。200
# 專有名詞，地名，「基利波」。0
# 專有名詞，人名，「麥基舒亞」，看撒母耳記上14:49。0
# 陽性名詞，「受膏者」，看利未記4:3。0
陽性複數名詞，「水」。 是 的停頓形，是冠詞 加名詞複揚。500
v6 陽性名詞，「光」。 係複數。100
# 專有名詞，族名，「亞捫」。100
# 陽性名詞，「事情、話語」。 是單陽附屬形； 是複數； 是複陽附屬形。500
