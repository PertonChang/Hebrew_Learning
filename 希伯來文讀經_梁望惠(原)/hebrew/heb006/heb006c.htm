第六章  詩篇105篇

生詞

          

v 表示第幾節，# 表示錄音帶未錄此字，* 表示動詞的這個形式在聖經中不存在；最後面的 0, 50, 100, 200, 500 分別代表它在聖經中出現 0-50, 50-100, 100-200, 200-500, 500 次以上；以下的詞類變化是為了方便練習解答查閱用，讀生詞時可先略過不看。

     
v1 動詞，「呼喊、求告、稱…為」。   是 Qal 祈使式複陽。500
陰性名詞，「作為、行為」。 是複陰加 3 單陽詞尾。0
v2 * 動詞，Pi'el「奏樂」。這個動詞只有 Pi'el 的形式； 是 Pi'el 祈使式複陽；* 是 Pi'el 完成式 3 單陽。0
動詞，「思想、默念」。    是 Qal 祈使式複陽；  是不定詞附屬形。0
v3 陽性名詞，「聖潔」。 是單陽加 3 單陽詞尾。200
# 陽性名詞，「文件、書」。 是單陽加 3 單陽詞尾。100
動詞，「喜樂、高興」。 是 Qal 未完成式 3 單陽，在此是祈願式（Jussive）。100
陽性名詞，「心」。其單陽附屬形也是 。500
* 動詞， Pi'el 「尋找」。這個動詞沒有 Qal 的形式； 是 Pi'el 分詞複陽附屬形； 是 Pi'el 祈使式複陽； 是 Pi'el 完成式 3 單陽。200
v4 動詞，「尋求、詢問」。 是 Qal 祈使式複陽。100
陽性複數名詞，「臉、前面」。 是複陽附屬形； 是複陽加 3 單陽詞尾。500
副詞，「總是、繼續」。100
v5 動詞，「記得、記念」。 是 Qal 祈使式複陽。100
關係詞。500
陽性名詞，「奇事、神蹟」。 是連接詞   加複陽加 3 單陽詞尾。這個名詞雖為陽性，但有陰性形式。0
* 動詞，「判斷、管理」。100
v6 陽性名詞，「判語、判斷」。 是複陽附屬形。200
陽性名詞，「口」。 是單陽附屬形， 和 都是單陽加 3 單陽詞尾。200
