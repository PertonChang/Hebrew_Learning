生詞

          

v 表示第幾節，# 表示錄音帶未錄此字，* 表示動詞的這個形式在聖經中不存在；最後面的 0, 50, 100, 200, 500 分別代表它在聖經中出現 0-50, 50-100, 100-200, 200-500, 500 次以上；以下的詞類變化是為了方便練習解答查閱用，讀生詞時可先略過不看。

     
v1 陽性名詞，「歌」。50
形容詞，陽性單數，「新的」。50
v2 * 動詞，Qal「有福的」；Pi'el「稱頌、祝福」。這個動詞的 Qal 只以被動分詞出現； 是 Pi'el 祈使式複陽。200
陽性名詞，「名字」。 係單陽加 3單陽詞尾。500
* 動詞，Pi'el「傳揚好消息」。這個動詞沒有 Qal 的形式； 是 Pi'el 祈使式複陽。0
介系詞，「從、出」。500
陽性名詞，「日子」。    是介系詞 加名詞單陽； 是介系詞 加名詞單陽。500
陰性名詞，「救恩、幫助」。 係單陰加 3 單陽詞尾。50
v3 動詞，Qal「數數目」；Pi'el「告訴、述說」。 是 Pi'el 祈使式複陽。100
陽性名詞，「重量、榮耀」。 係單陽加 3 單陽詞尾。200
v4 副詞，「很多，很」。200
# 專有名詞，人名，「撒迦利亞」。 是加上連接詞 。看列王紀下15:8。0
# 專有名詞，人名，「摩西」。 是加上連接詞 。看出埃及記2:10。500
v5 陽性名詞，「偶像、虛無」。 是複數。0
# 專有名詞，人名，「掃羅」。200
陽性名詞，「黃金」。 是介系詞 加名詞單陽。200
v6 # 陽性複數名詞，「臉、前面」。 是介系詞 加名詞複陽； 是複陽附屬形；是介系詞 加名詞複陽附屬形，它亦可作副詞，意思是「在…面前」； 是加 3單陽詞尾，意思是「在他面前」。500
陽性名詞，「力量、能力」。50
陰性名詞，「尊榮、華美」。 是連接詞 加名詞單陰。50
陽性名詞，「聖所」。 是介系詞 加名詞單陽加 3 單陽詞尾。50
# 陽性名詞，「聖潔」。200
# 形容詞，陽性單數，「聖潔的」。100
# 陽性名詞，「塔」。0
# 動詞，「騎」。 是 Qal 完成式 3 單陽。50
# 陰性名詞，「戰車」。0
